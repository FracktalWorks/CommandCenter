WhatsApp Message Manager

UI mockups for whatsapp_message_manager.md — the email playbook (triage → categories → drafts → digest → rules) applied to the founder's real inbox: WhatsApp. Visual language matches the control plane (cyan = primary/AI, WhatsApp green = channel/outbound, gold = VIP/commitments, violet = groups). Seven screens: connect, dashboard, conversation + copilot, categories, rules, digest, companion chat.

1 · Connect — two honest paths onto one pipeline

Both paths land in the same message store; everything downstream (triage, drafts, digest) is path-agnostic. The trade-offs are stated in the UI, not buried in docs.
